# LifeLimit
Life Limit  
あなたの人生の残り時間をナノ秒単位でお知らせします

## アプリケーション概要

生年月日と性別を設定すると残りの寿命がナノ秒単位で分かるアプリ  

## 背景

平均寿命と比べた時に自分の余命はどれくらいなのか。  
1分1秒を大切に生き抜くためにも時間を意識した生活をしたい。  
といった課題を解決するためのアプリ。

## ポイント

生年月日と性別を設定するだけで残りの寿命が分かる。  
壁紙を自由に設定できる。
